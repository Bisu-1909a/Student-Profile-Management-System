# Student Profile Management System

A professional command-line Student Profile Management System built using **Python**. This project demonstrates CRUD operations, file handling, object-oriented programming, authentication, attendance management, logging, and clean project architecture.

---

## Features

- 🔐 Admin Login Authentication
- ➕ Add Student
- 📋 View Students
- 🔍 Search Student
- ✏️ Update Student
- ❌ Delete Student
- 📅 Mark Attendance
- 📊 Student Statistics Dashboard
- 📝 Activity Logging
- ✅ Input Validation
- 📂 CSV-Based Data Storage

##  Installation

Clone the repository

```bash

git clone <https://github.com/Bisu-1909a/Student-Profile-Management-System.git>

```

Go into the project directory

```bash
cd Student-Profile-Management-System
```

Run the project

```bash
python main.py
```
